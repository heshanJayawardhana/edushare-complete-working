# EduShare MERN Project

This is a cleaned and organized client/server version of the EduShare project based on the original codebase structure you shared.

## Structure

- `client/` – React frontend in JavaScript/JSX
- `server/` – Express + MongoDB backend
- `start-all.js` – helper to start the full app from the project root

## Main working flows

- user registration and login
- profile loading
- resource listing with search/filter
- resource upload with file upload endpoint
- resource detail page
- ratings
- comments
- inquiries
- cart and checkout
- library view
- notifications
- student connections
- admin dashboard
